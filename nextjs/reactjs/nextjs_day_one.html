Question 2

How to set up a React project and render a simple component.

We can follow below steps:

  1. React works on node platform so fisrt we have to install node on machine, this will gives us npm (node package manager)
  2. Then we can create our application using command : "npx create-react-app hello-word"
  3. Now we can run the app, while inside root of the project folder, run "npm start"
  4. Now we can create a component inside this app:
    • create a file named Hello.js
    • Write a function to return what you want 
                          
                        import React from "react";
                        export default function HelloWorld(){
                            return(
                                
      Hello World
      
                            )
                        }
    • We can also keep this file inside "components" folder
    • Now our component is ready for render, we can import and call this inside our App component:
      • 
import React from "react";
import HelloWorld from './components/hello.js';
function App() {
  return (
    
      <HelloWorld />
      
  );
}

export default App;
